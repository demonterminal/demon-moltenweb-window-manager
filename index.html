Introduction

The Molten Window Manager enables you to create and manage draggable and resizable windows on your web pages. With this tool, you can customize window behavior, including minimizing, restoring, and setting window properties.

Installation

Include the Molten Window Manager script in your HTML file using the following CDN link:

<script src="https://demonterminal.github.io/demon-moltenweb-window-manager/demonservice.js"></script>

Basic Usage

To create a window, use the `window.open` method provided by the Molten Window Manager. Here's a basic example:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Molten Window Manager</title>
    <script src="https://demonterminal.github.io/demon-moltenweb-window-manager/demonservice.js"></script>
</head>
<body>
    <div id="desktop"></div>
    <div id="minimized-windows"></div>
    <script>
        const newWindow = window.open('https://example.com', 'My Window', 'width=400,height=300,icon=icon.png,winid=minimized-windows,target=_blank');
    </script>
</body>
</html>

Creating a New Window

Create a new window by calling the `window.open` function with the URL, name, and other parameters:

const newWindow = window.open('https://example.com', 'My Window', 'width=400,height=300,icon=icon.png,winid=minimized-windows,target=_blank');

Customizing Windows

You can customize your window's appearance and behavior using the following parameters:

  • url: The URL to be loaded in the window.
  • name: The title of the window.
  • width: The initial width of the window.
  • height: The initial height of the window.
  • icon: The URL of the icon to display in the window header.
  • winid: The ID of the element where minimized window icons will be placed.
  • target: The target attribute (e.g., _blank, _parent).

Handling Window Minimization

When a window is minimized, an icon will appear in the element specified by the winid parameter. Clicking the icon will restore the window.

Full Example

Here's a full example showing how to create multiple windows with different properties:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Molten Window Manager</title>
    <script src="https://demonterminal.github.io/demon-moltenweb-window-manager/demonservice.js"></script>
</head>
<body>
    <div id="desktop"></div>
    <div id="minimized-windows"></div>
    <script>
        // Open the first window
        window.open('https://example.com', 'First Window', 'width=400,height=300,icon=icon.png,winid=minimized-windows,target=_blank');

        // Open the second window with a different size
        window.open('https://example.org', 'Second Window', 'width=600,height=400,icon=icon.png,winid=minimized-windows,target=_blank');
    </script>
</body>
</html>
